# Contributing

Thanks for improving `public-surface-sweeper`.

## Development

1. Create a branch from `main`.
2. Install test dependencies with `python -m pip install -e ".[test]"`.
3. Run `python -m pytest`.
4. Keep new checks focused, documented, and covered by tests.

## Security

Do not open issues or pull requests containing secrets, credentials, private
customer data, or live tokens. Use placeholders in examples.
